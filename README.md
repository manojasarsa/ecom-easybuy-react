# Easy Buy
easYbuY is an fashion e-commerce application which is a one stop shop for all your fashion and lifestyle needs. 

## Features Present 
- Authentication
    - Login
    - Logout
    - Signup
- Product Listing
- Filter Products by
    - Category
    - Discount
    - Price
    - Rating
- Cart Management
- Wishlist Management
- Search by 
  - Product Name
- Single Product Page
- Alerts
- User Profile Page
- Order Summary
